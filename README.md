# Skills

These are the skills I personally use with my AI agents.

## Install a skill

List available skills in this repo:

```bash
npx skills add jarretthuang/skills --list
```

Install a specific skill from this repo:

```bash
npx skills add jarretthuang/skills --skill frontend-testing
```

Install globally instead of project-local:

```bash
npx skills add jarretthuang/skills --skill frontend-testing -g
```
